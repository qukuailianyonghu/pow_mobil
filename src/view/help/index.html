DCR挖矿配置详解

第一步要有一个接收DCR挖矿收益的钱包地址,可以通过以下途径获取：
全节点钱包（推荐）：https://www.decred.org/downloads/；网页钱包：https://www.decred.org/downloads/
如果你是技术人员的话，也可以使用命令行钱包，对应的git：https://github.com/decred/dcrwallet.git
1、使用显卡挖矿，目前支持DCR显卡挖矿的软件常用的有：gominer(官方维护)，sgminer,ccminer
gominer：由decred官方维护，对应的官方github地址：https://github.com/decred/gominer.git 根据文档构建好对应项目
通过命令 gominer -o stratum+tcp://dcr.2100pool.com:3252 -m 钱包地址.矿工号 -n x 启动挖矿程序

sgminer：A卡使用此软件效果更好，由于sgminer版本比较多，只有5.0以上的版本支持dcr，对应的github地址：https://github.com/tpruvot/sgminer.git
这个代码我们团队亲测过，可以正常使用，通过命令:sgminer -o stratum+tcp://dcr.2100pool.com:3252 -u username -p password 启动矿池

ccminer:由第三团队开发，对应的github地址：https://github.com/tpruvot/ccminer.git，构建之后可以通过命令:
ccminer -a decred -o stratum+tcp://dcr.2100pool.com:3252 -u 钱包地址.矿工号 -p x
建议使用官方支持挖矿软件，方面简单，出错好排查
使用矿机挖矿的话，需要我们自己手动配置矿机。目前支持DCR的ASIC矿机有：飞鱼D18、D9矿机（据听说神马要出44T的decred矿机）
下面以飞鱼D18示例，讲一下如何配置矿机：
1， 矿机通电，连接网络
2， 保证用于设置矿机的个人电脑与矿机在同一个域的局域网内
3， 运行ipreporter程序
4， 矿机通电后，应看到绿灯闪动，此时点击指示灯旁边的黑色按钮两秒，能看到在ipreporter上，弹出矿机的IP地址
6， 点击浏览器链接，打开飞鱼管理界面，输入账号密码（均为root）
7、矿池配置：币种选择DCR，点击提交，矿池地址填写前两个，一个首选（2100pool），一个备用。矿池地址：stratum+tcp://dcr.2100pool.com:3252 备选地址：stratum+tcp://us-west.luxor.tech:4445 用户填写自己的钱包地址，密码随便设置一个即可。钱包地址可在后面加 . 再加任意字母数字代表你的矿工名（如.11x）。Extranonce后面的yes不打勾
查看DCR矿机运行状态和每日收益，你可以关注我们矿池公众号（二维码在首页），随时随地查看自己的挖矿收益 和自己的矿机信息，我们也开通了掉线通知功能，关注我们公众号后，如果你的矿机因意外情况掉线我们会及时通知到您；
另外你也 可以通过访问2100Pool（https://www.2100POOL.com）官网，点击首页头部的“查看收益”按钮，输入您的地址，进行查看。

常见问题

  • 比特币BTC：挖矿设备为比特币矿机，例如蚂蚁S9、阿瓦隆A741
  • 莱特币LTC：挖矿设备为莱特币矿机，例如蚂蚁L3+、阿瓦隆A471
  • 以太坊ETH、以太坊经典ETC：挖矿设备为显卡GPU，例如470、570，挖ETH显存3G及以上，挖ETC显存2G及以上。
  • 零币ZEC：挖矿设备为显卡GPU，例如1060，显存2G及以上
  • 云储币SC：挖矿设备为显卡GPU，建议与ETH或ETC一起进行双挖
  • DCR:支持显卡和矿机，飞鱼D18，D9
若想获得停机通知则只需关注我们矿池的官方微信，并绑定对应币种的挖矿地址即可(对应官网微信在首页)
每日付款一次，每日支付的是昨天早8点到今天早10点之间的收益。 支付时间：10：00-16：00 注意：每天8点结算时账户余额达到起付数量的自动结算，当日支付；不够起付数量的当日不结算不支付。 支付规则 : 0.1DCR 起付
我们打币给用户包含三个环节：2100POOL发出交易，区块链打包确认，钱包或交易平台上账，三个环节都完成则打币成功。如果您没有收到币，请按以下情况确认哪个环节未完成，然后再找相应的人员处理:
1. 2100POOL这边有付款记录，而且交易一列已有交易号/txid，交易平台或钱包没有充值记录说明打币的前两个环节已完成，不到账的请联系您所在的交易平台。交易平台一般对一笔充值都有确认数要求，达到某个值才给上账，点击2100POOL付款记录的交易号链接，查看确认数，确认数不够的等够了再查看账户。如果确认数够了但没给充值，联系交易所提交工单给您把这笔充值上账。 如果您使用的是本地钱包，如DCR客户端钱包 请保证钱包同步到最新区块。
2. 2100POOL这边有付款记录，但交易一列现实正在处理区块链未打包确认，有时区块链会发生拥堵，等待确认时间较长，请耐心等待确认。
3. 2100POOL这边没有付款记录:
（1）早10点时账户余额不够起付，当日无法结算支付；
（2）早10点时账户余额够起付，请耐心等待，当日肯定会支付，支付时间早8点到下午4点
均不是以上情况，早10点时账户余额够起付，且已经超过下午4点，请通过邮箱、微信群或电话联系我们。
1. 如果账户余额够起付，等第二天自动支付
2. 如果账户余额未达到起付标准，请在2100POOL官网客服中心，提交工单。按照如下格式申请转出余额：“币种+地址，因×××××××（叙述原因），申请转出账户余额。”（注意：余额只能转出到挖矿时候绑定的钱包地址 账户余额转出申请只在每月底最后一个工作日处理一次，网络转账手续费用户自己承担。
如若要更换挖矿地址，则只需在矿机端从新配置新的挖矿地址即可，原来的地址上的币如果大于0.1起付金额，2100pool会在第二天 的10：00到16：00之间把币打到原来的地址上，如果原来地址剩余的币小于0.1起付金额，则可以通过矿池官方微信联系我们的工作人员 说明原因，我们会及时把币转到原来的地址，也可以通过提交工单的形式说明
